# sw - a simple, precise stopwatch

Precisely record times to the 1/100th of a second.

## Installation

`make && make install`

## Usage

`sw`

`SPC` to pause/resume.

`r` to reset.

`s` to save.

`q` to quit.

`sw -s` to save time to `~/.sw/savedtime`

`sw -r` to restore time from `~/.sw/savedtime`
